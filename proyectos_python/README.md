# Proyectos Python

Esta carpeta contiene mis proyectos de Python.

## Estructura

- `proyecto_1/` - Descripción del primer proyecto
- `proyecto_2/` - Descripción del segundo proyecto
- etc.

## Instrucciones para añadir proyectos

1. Crea una subcarpeta con el nombre del proyecto
2. Incluye un `README.md` con la descripción del proyecto
3. Coloca el código Python en la carpeta
4. Actualiza este archivo con la información del nuevo proyecto
